# erzoc.dev

Website for the fans, folk, and friends back home.

Personal site and blog — built with [Astro](https://astro.build), deployed on GitHub Pages.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build & deploy

```bash
npm run build
```

Pushes to `main` build and deploy automatically via GitHub Actions.
